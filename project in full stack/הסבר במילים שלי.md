# מערכת ניהול קמפיין פרסומי - Aniti Jewelry

## תיאור הפרויקט
הפרויקט הוא מערכת אינטראקטיבית לניהול קמפיין פרסומי עבור חברת התכשיטים "Aniti Jewelry". המערכת כוללת דפים שונים לניהול קמפיינים, יצירת באנרים, דפים שיווקיים ודפי נחיתה, ומשתמשת רק בטכנולוגיות צד לקוח.

## טכנולוגיות בשימוש
- HTML5
- CSS3
- JavaScript
- Local Storage

## תכולת הפרויקט
1. **דף כניסה (index.html)**:
   - טופס התחברות עם שדות לשם משתמש וסיסמה.
   - קישור לדף הרשמה למשתמשים חדשים.
   - קישור לשחזור סיסמה.

2. **דף ראשי (main.html)**:
   - תפריט ניווט בין חלקי המערכת.
   - סקירה כללית של הקמפיין הנוכחי.

3. **עורך באנרים (banner_editor.html)**:
   - ממשק ליצירת באנר לאורך ולרוחב.
   - שדות עריכה לטקסט, צבעים, גופנים וגדלים.
   - תצוגה מקדימה בזמן אמת של הבאנר.

4. **עורך דף שיווקי (marketing_editor.html)**:
   - תבנית בסיסית לדף שיווקי הנשלח במייל.
   - אפשרויות לעריכת תוכן, תמונות, צבעים וסגנון.
   - תצוגה מקדימה של הדף השיווקי.

5. **עורך דף נחיתה (landing_page_editor.html)**:
   - ממשק לעיצוב ועריכה של דף נחיתה.
   - אפשרויות לעריכת כותרות, פסקאות טקסט, תמונות וכפתורי פעולה.
   - תצוגה מקדימה בזמן אמת של דף הנחיתה.
   - אפשרות להוספת טופס איסוף פרטים (שם ואימייל).

## התקנה והרצה
1. פתחו את הפרויקט בעורך הקוד המועדף עליכם.
2. ודאו שכל הקבצים והתיקיות נמצאים במקומות הנכונים.
3. פתחו את הקובץ `index.html` בדפדפן.
4. בצעו הרשמה או התחברות לפי ההוראות בדף.

## שמירת נתונים
כל המידע נשמר ב-Local Storage של הדפדפן, כולל פרטי משתמשים, באנרים, דפים שיווקיים ודפי נחיתה.

## עיצוב רספונסיבי
האתר מתאים למגוון מכשירים, כולל טאבלטים ומובייל, ומציע חוויית משתמש נוחה בכל פלטפורמה.

## ולידציה
המערכת מבצעת ולידציה של קלטי המשתמש בצד הלקוח עם JavaScript, כולל טיפול בשגיאות והצגת הודעות מותאמות.
